# Usage

## Installation

To install hplib:

```console
(.venv) $ pip install hplib
```

## import function
```console
from hplib import hplib as hpl
```

## load database

To load the whole database use {py:func}`hplib.hplib.load_database` function:
```{eval-rst}
.. autofunction:: hplib.hplib.load_database
```
